Вариант: № 1

## Описание проекта

Разработано программное обеспечение для мобильного робота TurtleBot3 (модель Burger) 
на базе фреймворка ROS 2 Humble. Проект включает два основных модуля:

1. **Нода движения по геометрической траектории** — автоматическое прохождение 
   квадрата со стороной 1.0 м по часовой стрелке.
2. **Клиент автономной навигации Nav2** — программная отправка последовательности 
   из трёх целевых точек (включая проблемную цель вплотную к препятствию) с 
   обработкой результатов.

## Технологии

- **ОС:** Ubuntu 22.04 LTS (через WSL2 на Windows)
- **Фреймворк:** ROS 2 Humble Hawksbill
- **Симулятор:** Gazebo Classic 11
- **Стек навигации:** Nav2 + SLAM Toolbox + Cartographer
- **Язык:** Python 3.10 (rclpy)
